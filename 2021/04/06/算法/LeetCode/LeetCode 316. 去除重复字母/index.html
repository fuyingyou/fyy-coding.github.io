LeetCode 316. 去除重复字母

题目 去除重复字母

给你一个字符串 s ，请你去除字符串中重复的字母，使得每个字母只出现一次。需保证 返回结果的字典序最小（要求不能打乱其他字符的相对位置）。

示例1

输入：s = “bcabc”
输出：“abc”

示例2

输入：s = “cbacdcbc”
输出：“acdb”

数据范围

1 <= s.length <= 104
s 由小写英文字母组成

来源：力扣（LeetCode）
https://leetcode-cn.com/problems/remove-duplicate-letters/
